# 📝 人類文本收集說明

## 🎯 回答您的問題

### 1. 需要收集幾篇人類文本？

**建議數量：**
- **最少：100-200 篇**
- **理想：500-1000 篇**（與 AI 生成文本數量相當）

**原因：**
- 平衡資料集（AI vs Human 比例接近 1:1）
- 確保每個主題都有足夠樣本
- 提高模型泛化能力

### 2. 需要哪些主題？

**請參考 `data/topics.txt` 中的 15 個主題：**

1. 人工智慧的未來發展
2. 環境保護與永續發展
3. 遠距工作的優缺點
4. 教育改革的必要性
5. 科技對生活的影響
6. 健康飲食的重要性
7. 閱讀習慣的培養
8. 旅遊的意義與價值
9. 音樂對情緒的影響
10. 運動與身心健康
11. 網路隱私與安全
12. 社交媒體的影響
13. 氣候變遷的挑戰
14. 創意寫作的重要性
15. 時間管理技巧

**建議：**
- 每個主題收集 **10-20 篇**人類文本
- 確保主題與 AI 生成文本對應
- 收集多樣化的寫作風格

### 3. 放在哪個資料夾？

**選項 1：合併到主資料集（推薦）**

將人類文本合併到 `AI_Human.csv`，格式：

```csv
text,generated
"人類撰寫的文本內容...",False
"另一篇人類文本...",False
```

**選項 2：單獨檔案**

儲存在 `data/human_texts.csv`，格式：

```csv
text,topic,label
"人類撰寫的文本內容...","人工智慧的未來發展","Human"
"另一篇人類文本...","環境保護與永續發展","Human"
```

## 📊 資料格式要求

### CSV 格式
- **text**: 文本內容（200-800 字為佳）
- **generated**: `False` 或 `0`（表示人類撰寫）
- **topic** (可選): 主題標籤

### 範例

```csv
text,generated,topic
"人工智慧在近年來快速發展，從語音助手到自動駕駛，AI 技術已經深入我們的生活。然而，這種快速發展也帶來了許多挑戰和思考。首先，我們需要考慮 AI 對就業市場的影響。許多傳統工作可能會被自動化取代，但同時也會創造新的工作機會。其次，AI 的倫理問題也值得我們深思。如何確保 AI 系統的公平性和透明度，避免偏見和歧視，是我們必須面對的重要課題。",False,"人工智慧的未來發展"
```

## 🔍 文本來源建議

1. **學術論文摘要**（需注意版權）
2. **學生作業**（確保未使用 AI）
3. **個人部落格**（原創內容）
4. **論壇長文**（Reddit, PTT 等）
5. **讀者投書**（新聞評論）

## ⚠️ 注意事項

1. ✅ **版權**：確保有權使用收集的文本
2. ✅ **品質**：避免包含 AI 生成內容
3. ✅ **長度**：與 AI 文本長度相似（200-800 字）
4. ✅ **多樣性**：收集不同寫作風格
5. ✅ **主題對應**：與 AI 生成文本主題一致

## 🚀 使用方式

收集完成後：

1. **如果合併到 AI_Human.csv**：
   ```bash
   # 直接使用現有訓練腳本
   python train_pipeline/train_svm.py --data AI_Human.csv
   ```

2. **如果是單獨檔案**：
   ```python
   # 合併資料
   import pandas as pd
   
   ai_df = pd.read_csv('AI_Human.csv')
   human_df = pd.read_csv('data/human_texts.csv')
   
   # 轉換標籤
   human_df['generated'] = False
   
   # 合併
   combined_df = pd.concat([ai_df, human_df[['text', 'generated']]], ignore_index=True)
   combined_df.to_csv('AI_Human_combined.csv', index=False)
   ```
